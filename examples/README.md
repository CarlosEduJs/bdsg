# Examples

This directory contains practical examples demonstrating how to use bdsg.

## Examples

| Example | Description |
|---------|-------------|
| [color-palette.ts](./src/color-palette.ts) | Generate accessible color palettes |
| [typography-scale.ts](./src/typography-scale.ts) | Create typography scales with musical ratios |
| [spacing-scale.ts](./src/spacing-scale.ts) | Generate spacing values (Fibonacci, linear) |
| [shadows.ts](./src/shadows.ts) | Create Material Design shadows |
| [wcag-compliance.ts](./src/wcag-compliance.ts) | Check and fix WCAG contrast |
| [design-tokens.ts](./src/design-tokens.ts) | Export complete design token system |

## Running Examples

```bash
cd examples

# Run individual examples
bun run palette      # Color palette generation
bun run typography   # Typography scale
bun run spacing      # Spacing scale
bun run shadows      # Shadow generation
bun run wcag         # WCAG compliance
bun run tokens       # Complete design tokens
```
